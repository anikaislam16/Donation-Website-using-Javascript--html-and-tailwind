Important FAQ

What is Document Object Model?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It defines the logical structure of documents and the way they can be accessed and manipulated by programming languages like JavaScript.

How do you select an element from the DOM?

To select an element from the DOM, you use JavaScript's document object and various methods it provides. The most commonly used method is getElementById(), which selects an element based on its unique id attribute. Another popular method is querySelector(), which allows you to select the first matching element based on CSS selectors like element tags, classes, or IDs. For example, you can use document.querySelector(".className") to select an element by its class or document.querySelector("#idName") to select an element by its ID. Additionally, there are other methods like getElementsByClassName() and getElementsByTagName() that return collections of elements based on their class or tag. These methods are useful for manipulating specific parts of a webpage, such as changing text, updating attributes, or responding to user events like clicks or form submissions.

What is event delegation in the context of DOM and why it is useful?

Event delegation is a technique in the DOM that attaches a single event listener to a parent element instead of multiple listeners to child elements. It leverages event bubbling, allowing the parent to identify which child triggered the event using event.target.

The main advantages of event delegation include reduced memory usage by minimizing event listeners, automatic handling of new child elements, and cleaner code management. This makes it easier to maintain, particularly when dealing with multiple similar elements.

How do you manipulate an element's attributes and styles using DOM?

To change an element's attributes, you can use methods like setAttribute(), getAttribute(), or directly access properties of the element.

Examples using setAttribute(), getAttribute():

const element = document.getElementById('myElement');
element.setAttribute('src', 'newImage.png'); // Change the source of an image
const altText = element.getAttribute('alt'); // Get the alt text of the image

Examples using modifying style property

element.style.color = 'red'; // Change text color to red
element.style.backgroundColor = 'blue'; // Change background color to blue
